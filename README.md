# Market Tracker

## Project Description
Market Tracker is a comprehensive tool designed to monitor and analyze market trends in real time. By utilizing cutting-edge APIs and data processing techniques, this project enables users to gain insights into market conditions and make informed decisions.

## Features
- **Real-time Data Monitoring**: Pulls the latest market data using the AlphaVantage API.
- **Data Visualization**: Visualize market trends using Google Sheets and Looker Studio.
- **Excel Integration**: Allows users to export data to Excel for further analysis.
- **Hourly Updates**: Automated hourly market tracking via GitHub.
- **User-Friendly Interface**: Simple and intuitive interface for easy navigation and usage.

## Technologies Used
- **Python**: The primary programming language for data processing and API integration.
- **AlphaVantage API**: Source for real-time market data and stock information.
- **Excel**: For data storage, analysis and reporting.
- **Google Sheets**: For live data integration and collaboration.
- **Looker Studio**: For advanced data visualization and interactive dashboard creation.
- **GitHub**: For version control, automation, and hourly market tracking updates.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Omshekhar/market_tracker.git
   ```
2. Navigate into the project directory:
   ```bash
   cd market_tracker
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Set up your AlphaVantage API key in your environment variables.

## How It Works
1. Python script fetches real-time market data from AlphaVantage API
2. Data is stored and processed in Excel files
3. Google Sheets automatically syncs live data from GitHub
4. Looker Studio creates interactive dashboards from the data
5. GitHub automation runs hourly to update market data
